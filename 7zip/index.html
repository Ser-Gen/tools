Supported File Types

  • 7z (.7z): 7-Zip archive format.
  • APFS (.apfs): Apple File System image.
  • APM (.apm): Apple Partition Map.
  • AR (.ar .a .deb .udeb .lib): Unix archiver format.
  • ARJ (.arj): Arj archive format.
  • Base64 (.b64): Base64 encoded file.
  • BZIP2 (.bz2 .bzip2 .tbz2 .tbz): BZIP2 compression algorithm.
  • CAB (.cab): Microsoft Cabinet archive format.
  • CHM (.chm .chi .chq .chw): Microsoft Compiled HTML Help format.
  • Compound (.msi .msp .doc .xls .ppt): Compound file format.
  • COFF (.obj): Common Object File Format.
  • CPIO (.cpio): Unix cpio archive format.
  • CramFS (.cramfs): Compressed ROM File System image.
  • DMG (.dmg): Apple Disk Image format.
  • ELF (.elf): Executable and Linkable Format.
  • Ext (.ext .ext2 .ext3 .ext4 .img): Linux Extended File System image.
  • FAT (.fat .img): File Allocation Table disk image.
  • FLV (.flv): Flash Video format.
  • GPT (.gpt .mbr): GUID Partition Table or Master Boot Record.
  • GZIP (.gz .gzip .tgz .tpz .apk): GNU ZIP compression algorithm.
  • HFS (.hfs .hfsx): Hierarchical File System image.
  • HXS (.hxs .hxi .hxr .hxq .hxw .lit): Microsoft Help 2 file format.
  • IHex (.ihex): Intel Hex format.
  • ISO (.iso .img): ISO disc image format.
  • LP (.lp .img): LP disc image format.
  • LZH (.lzh .lha): Lempel-Ziv-Huffman archive format.
  • LZMA (.lzma): LZMA compression algorithm.
  • LZMA86 (.lzma86): LZMA86 compression algorithm.
  • MBR (.mbr): Master Boot Record.
  • MachO (.macho): Mach-O Executable format.
  • MsLZ (.mslz): Microsoft LZ algorithm.
  • MUB (.mub): MUB partition format.
  • NTFS (.ntfs .img): New Technology File System image.
  • NSIS (.nsis): Nullsoft Scriptable Install System format.
  • PE (.exe .dll .sys): Portable Executable format.
  • PPMd (.pmd): PPMd compression algorithm.
  • QCOW (.qcow .qcow2 .qcow2c): QEMU Copy-On-Write format.
  • RAR (.rar .r00): RAR archive format.
  • RAR5 (.rar .r00): RAR5 archive format.
  • RPM (.rpm): RPM Package Manager format.
  • SWF (.swf): Small Web Format.
  • SWFc (.swf): SWF Compressed format.
  • Sparse (.simg .img): Sparse disk image format.
  • Split (.001): Split file format.
  • SquashFS (.squashfs): Squash File System format.
  • TAR (.tar .ova): Tape Archive format.
  • TE (.te): Text Encoding format.
  • UEFIc (.scap): UEFI capsule format.
  • UEFIf (.uefif): UEFI firmware format.
  • UDF (.udf .iso .img): Universal Disk Format.
  • VDI (.vdi): Virtual Disk Image format.
  • VHD (.vhd): Virtual Hard Disk format.
  • VHDX (.vhdx .avhdx): Virtual Hard Disk 2 format.
  • VMDK (.vmdk): Virtual Machine Disk format.
  • WIM (.wim .swm .esd .ppkg): Windows Imaging Format.
  • XAR (.xar .pkg .xip): XAR archive format.
  • XZ (.xz .txz): XZ format.
  • Z (.z .taz): Unix compress format.
  • ZIP (.zip .z01 .zipx .jar .xpi .odt .ods .docx .xlsx .epub .ipa .apk .appx): ZIP archive format.
  • ZSTD (.zst .tzst): Zstandard compression algorithm.
